# Mike Mancour artist legacy site bundle

This bundle contains a premium-style Next.js starter for mikemancour.com with:

- StarSpangledBanner set as the hero image
- real archive photos included in `/public/images`
- polished Home, About, Listen, Watch, and Timeline pages
- a simple email button wired to `mmanco01@msn.com`
- a stronger official timeline/story spine based on the confirmed public chronology

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`
